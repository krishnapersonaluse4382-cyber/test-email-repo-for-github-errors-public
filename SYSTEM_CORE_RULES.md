## 1. INFRASTRUCTURE & CREDENTIALS
- **Database**: Supabase (Project: zukjamcmpsjjarhjftpy)
- **Table Schema**: `email_logs`
    - `email` (text): The recipient address.
    - `campaign_id` (text): Unique ID for the current batch (e.g. `BATCH_001`).
    - `status` (text): `SENDING_NOW`, `SENT`, or `ERROR`.
    - `body` (text): The personalized copy generated by the "Brain".
